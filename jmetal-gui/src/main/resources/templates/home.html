Welcome to the jMetal Graphic User Interface

jMetal, which stands for Metaheuristic Algorithms in Java, is an object-oriented Java-based framework for multi-objective optimization with metaheuristics.

The object-oriented architecture of the framework and its included features allows you to: experiment with the provided classic and state-of-the-art techniques, develop your own algorithms, solve your optimization problems, integrate jMetal in other tools, etc.
